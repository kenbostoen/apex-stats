# apex-stats
apex game player statistic viewer

(images below)

# Technologies used
Frontend: [VueJS](https://vuejs.org/)

Backend: [Java springboot](https://spring.io/projects/spring-boot)


# How to run

Frontend:
```
  npm install
  npm run serve
  ```

Backend: 
```
  import maven dependencies
  mvn spring-boot:run
```
